# vue-cli-template
vue cli 构建的 template


## 基础功能
- cdn
- 打包分析
- 单测
## 命令
```
// 启动
npm run serve
// 构建
npm run build
// 单测
npm run test:unit
// 打包分析
npm run analyzer
```
